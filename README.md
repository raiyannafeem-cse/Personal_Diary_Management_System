## 📌 1. Project Description
The **Personal Diary Management System** is a console-based application developed using the C programming language.  
It allows users to store, retrieve, and search personal diary entries using file handling.

The system is designed strictly based on **course lecture materials**, ensuring all implemented features follow concepts taught in class.

---

## 🎯 2. Problem Statement
Managing personal notes manually can be inefficient and unorganized.  
This project solves the problem by providing a simple digital system where users can:
- Save diary entries
- View stored notes
- Search specific information quickly

---

## 🎯 3. Objectives
- Implement **file handling operations**
- Use **functions for modular programming**
- Apply **string manipulation techniques**
- Build a **menu-driven program**
- Ensure **data persistence using files**

---

## ⚙️ 4. Functional Requirements
The system provides the following functionalities:

- ✍️ Add new diary entry  
- 🔍 Search entries by keyword/date  
- 📖 Edit or delete entries  
- ❌ Exit the application  

---

## 🛠️ 5. Non-Functional Requirements
- Simple and user-friendly interface  
- Fast execution (low memory usage)  
- Reliable file storage  
- Easy to maintain and modify  

---

## 🧠 6. Concepts Used (From Lecture)
- File Handling (`fopen`, `fprintf`, `fgets`, `fclose`)
- Strings (`strcpy`, `strcmp`, `strstr`)
- Functions (modular programming)
- Loops (`while`, `for`)
- Conditional Statements (`if-else`)
